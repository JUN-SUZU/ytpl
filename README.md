# @jun-suzu/ytpl

A light-weight ytpl parser for Node.js.
Simple and easy to use.
You can get the playlist information by providing the playlist URL or ID.

[Original ytpl](https://github.com/TimeForANinja/node-ytpl) is a great library, but it has some issues with the latest YouTube changes and is no longer maintained. This library is a fork of the original ytpl, with some modifications to make it work with the latest YouTube changes.
